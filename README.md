# Wanderlust Project 🌍

A full-stack travel listing web application inspired by Airbnb.  
Users can create, edit, and manage travel listings with image uploads and authentication features.

## Tech Stack
- Node.js
- Express.js
- MongoDB
- EJS
- Cloudinary

## ✨ Features
- User Authentication
- CRUD Operations
- Image Uploads
- Responsive Design
- Reviews & Ratings
